Multiplayer Game Project

My first attempt at making a game with working multiplayer.

Project Overview

The goal of this project was to design and develop a multiplayer game client application using the Unreal Engine framework. The secondary aim was to use Unreal C++ over blueprints to create the game functionality.

A cooperative platformer was the chosen style of game for this project. The Stylized Egypt from the Unreal Engine Marketplace was used as the visual basis for the game.
