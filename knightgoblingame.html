Knight Versus Goblins Game

My first game made using only C++, without a game engine like Unreal Engine.

Project Overview

This was the first game I made in C++ that wasn't aided by the use of a game engine. It is made using the Raylib library, and free assets from Itch.io. The intent is to build a more robust Top-Down Action RPG style game, once I become more proficient in C++, using the basic framework already established in this project.

The goal of this project was to create a simple game using the basics of C++ programming such as loops, arrays, functions, and pointers etc., that could then easily be built upon at a later date.
